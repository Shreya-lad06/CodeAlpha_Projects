# 📊 Unemployment Analysis in India using Python

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Science Internship**.

The objective of this project is to analyze unemployment trends in India, identify the impact of the Covid-19 pandemic on employment, explore regional and seasonal patterns, and generate actionable insights using data visualization and statistical analysis techniques.

---

## 🎯 Objectives

* Analyze unemployment rate data across different regions of India.
* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA).
* Investigate the impact of Covid-19 on unemployment rates.
* Identify seasonal and regional unemployment trends.
* Build interactive visualizations and dashboards.
* Generate insights and policy recommendations.

---

## 📂 Dataset Information

The dataset contains information related to:

* Region (State/UT)
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed Population
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## 📈 Project Workflow

### 1. Data Collection

* Imported unemployment dataset.
* Loaded data using Pandas.

### 2. Data Cleaning

* Removed missing values.
* Converted date columns to datetime format.
* Renamed columns for easier analysis.

### 3. Exploratory Data Analysis

* Descriptive statistics.
* Distribution analysis.
* State-wise unemployment comparison.
* Monthly trend analysis.

### 4. Covid-19 Impact Analysis

* Compared unemployment before and during Covid-19.
* Identified unemployment spikes during lockdown periods.

### 5. Regional Analysis

* Urban vs Rural unemployment comparison.
* State-wise unemployment ranking.

### 6. Seasonal Trend Analysis

* Month-wise unemployment behavior.
* Pattern identification across different periods.

### 7. Interactive Dashboard

* State-wise unemployment trends.
* Top affected states visualization.
* Labour participation analysis.

### 8. Unemployment Forecasting

* Future unemployment trend prediction using Linear Regression.

---

## 📊 Key Findings

* Covid-19 caused a significant increase in unemployment rates across India.
* Urban regions experienced greater employment disruptions compared to rural areas.
* Certain states consistently showed higher unemployment rates.
* Labour participation rate and unemployment rate exhibit noticeable relationships.
* Seasonal fluctuations suggest that economic activities influence employment patterns.
